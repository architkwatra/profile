Peer To Peer with Centralized Index

Designed and develped a peer to peer architecture for transferring files among different peers. The architecture consists of one server and multiple peers/clients and any client could upload a file it has while any other client could download it from that peer. The server maintains a list of all the available files with their corresponding owners and any peer could use that list to get the information on the available files.
